# Program Command Center public demo

This demo shows a synthetic lead pipeline, a local HTTP API, guarded state transitions, request-id idempotency, and append-only audit evidence. It does **not** include campaign configuration, actual parents, client data, payments, Meta/Resend connections, tracking links, production database logic, or the private product source.

Run it with `npm run demo:command-center`, then open `http://localhost:4173`.
